🛍️ Análise de Desempenho de Lojas - Alura Challenge
Este projeto tem como objetivo analisar o desempenho de quatro lojas e determinar, com base em dados, quais delas devem ser vendidas . A análise considera diversos critérios como faturamento, avaliações de clientes, mix de produtos, frete médio e abrangência geográfica.

📌 Objetivos
Avaliar o desempenho financeiro de cada loja
Analisar o comportamento de vendas por categoria
Observar as avaliações dos consumidores
identificar produtos com maior e menor giro
Avaliar o custo médio de frete
Verifique a distribuição distribuída de cada loja
Indicar qual loja apresenta menor potencial e deve ser vendida
🧪 Tecnologias Utilizadas
Python
Pandaspara análise de dados
Matplotlibe Seabornpara vídeos
Jupyter Notebook
Markdownpara documentação
📈 Resultados Principais
Loja 1 : Maior faturamento total
Loja 3 : Melhor avaliação média dos clientes
Loja 2 : Forte em nichos específicos, mas com baixa abrangência geográfica
Loja 4 : Apesar da maior cobertura nacional, apresenta o menor desempenho geral
🔎 Para detalhes completos, consulte o Relatório Final

📤 Conclusão
A Loja 4 foi indicada como a menos promissora do portfólio, enviando a recomendação principal sua venda estratégica .
